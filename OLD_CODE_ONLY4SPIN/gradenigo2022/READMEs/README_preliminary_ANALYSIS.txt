The scripts compute_logarithmic_averages.sh (generated by cla_gen.sh for each size ---> change NPT in the script) 
and logarithmic_windows_average.awk must be copied inside each sample and runned. 

logarithmic_windows_average.awk is an awk script that for a given temperature
of the sample (one of the NPT) computes the total mean energy and specific heat (its variance)
of the system in log_2 time, and similarly for the acceptation rate (and its variance).
NB: One has to set the Size (NSPIN) of the simulation before running it.

compute_logarithmic_averages.sh is a bash script that 
>> creates a directory ANALYSIS inside each sample
>> takes in input temperature, energy and acceptation rate (for each printed step) from the files
   parallel_tempering*.dat for each of the 2 (or 4) replicas
>> runs the awk script for each temperature and for each of the 2 (or 4) replicas
>> output files:
   > s*acc_rate_j contains acc rates (and error, errE) at log time for the replica * at the temperature j 
   > s*tempj contains energy (and error) at log time for the replica * at the temperature j 
   
   > calore_specifico_s* contains specific heat (calculated as errE/T^2) for each of the replicas, 
     using the last (thermalized) value of the energy variance of each of the NPT replicas
   > calore_specifico_all contains calore_specifico_s* pasted together
   > calore_specifico_media contains the mean specific heat averaged over the replicas *
     (and its error)
   > last three passages equal for energy with the files energy_s*, energy_all and energy_media
   
NB: BEFORE PLOTTING the values of the averaged acc_rate must be divided by NSPIN and their error by NSPIN^2

The script prova_term.sh (launched by run_prova_term.sh) is the last part of 
compute_logarithmic_averages.sh and can be adapted to compute specific heat
and energy using the averages made over different logarhtmic windows, 
io order to compare them to see if the system has thermalized.
For 2^18 MCsteps --- 196608
    2^19 MCsteps --- 393216
    2^20 MCsteps --- 786432
    2^21 MCsteps --- 1.57286e+06
This numbers are the averaged times: the sum of all the numbers (times) in a logathmic 
window (eg between 2^17 and 2^18) divided by their number (eg 2^18-2^17=2^17)
